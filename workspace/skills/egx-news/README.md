## 🔄 كيف يعمل النظام؟

### 1️⃣ جلب الأخبار (`fetch_news.py`)

```bash
python3 scripts/fetch_news.py --live
```

| الخطوة | الوصف |
|--------|-------|
| 1. جلب RSS | يجلب أخبار من Google News RSS (مجاني) |
| 2. Parse XML | يحلل الـ XML ويستخرج العناوين والمصادر |
| 3. Double-Check | يفلتر الأخبار حسب: المصدر الموثوق، كلمات طويل الأجل، PE Ratio |
| 4. تخزين | يحفظ الأخبار الموثوقة في SQLite |
| 5. تحديد الأولوية | high / medium / low |

**الخيارات:**
```bash
--live      # جلب أخبار حية من Google News
--raw       # طباعة الأخبار الخام للـ LLM
--days 7    # أخبار آخر 7 أيام
--symbol CIB # أخبار سهم معين
```

---

### 2️⃣ التقرير اليومي (`daily_report.py`)

```bash
python3 scripts/daily_report.py
```

| الخطوة | الوصف |
|--------|-------|
| 1. قراءة DB | يقرأ الأخبار الموثوقة من آخر 24 ساعة |
| 2. ترتيب | يرتب حسب الأولوية (high أولاً) |
| 3. إنشاء Markdown | ينشئ تقرير منسق |
| 4. حفظ | يحفظ في `reports/daily-report-YYYY-MM-DD.md` |
| 5. Telegram | يرسل ملخص + الملف على Telegram |
| 6. تحديث DB | يعلّم الأخبار كـ "processed" |

**الخيارات:**
```bash
--no-send   # بدون إرسال Telegram (للاختبار)
```

---

### 3️⃣ التقرير الأسبوعي (`weekly_report.py`)

```bash
python3 scripts/weekly_report.py
```

| الخطوة | الوصف |
|--------|-------|
| 1. قراءة DB | يقرأ أخبار آخر 7 أيام |
| 2. إحصائيات | يحسب: أهم المصادر، أكثر الأسهم ذكراً، متوسط PE |
| 3. تحليل | يقترح توزيع الاستثمار الشهري |
| 4. إنشاء تقرير | تقرير شامل مع جداول |
| 5. Telegram | يرسل التقرير كاملاً |

---

### 4️⃣ إرسال Telegram (`send_telegram.py`)

```bash
# رسالة بسيطة
python3 scripts/send_telegram.py "رسالة اختبار"

# رسالة مع ملف
python3 scripts/send_telegram.py --file report.md "ملخص التقرير"

# تلخيص تلقائي للملف ✨ جديد
python3 scripts/send_telegram.py --file report.md --summarize

# رسالة عاجلة
python3 scripts/send_telegram.py --priority high "تنبيه مهم!"
```

## 📊 قاعدة البيانات (SQLite)

**جدول `news`:**

| العمود | النوع | الوصف |
|--------|-------|-------|
| id | INTEGER | معرف فريد |
| date | TEXT | تاريخ الخبر |
| title | TEXT | عنوان الخبر |
| content | TEXT | محتوى الخبر |
| source | TEXT | المصدر |
| symbol | TEXT | رمز السهم (CIB, EGX30, etc.) |
| pe_ratio | REAL | مكرر الربحية |
| trusted | INTEGER | 0=غير موثوق، 1=موثوق |
| priority | TEXT | high/medium/low |
| processed | INTEGER | 0=جديد، 1=تم معالجته |
